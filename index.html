Como Ingressar

Faça o ENEM

Participe da ultima edicao do ENEM. Sua nota no ENEM sera usada para o processo seletivo.

Resultado

Após o fechamento das inscrições, vocé deve acessar o site do Sisu para verificar se foi aprovado para o curso desejado.

Inscrição no SISU

Selecione o curso "Tecnologia em Sistemas para Internet" no IFSP - Campus Itapetininga.

Matricula

Caso seja selecionado, vocé precisará realizar a matrícula conforme as orientações do câmpus.
